## 📌 Project Overview

This project analyses academic performance data of **25,000 students** to uncover patterns and insights related to subject scores, gender equity, study habits, and grade distribution. The entire workflow — from raw data to interactive dashboard — was completed using **Microsoft Excel**.

---

## 🎯 Objectives

- Clean and prepare raw student performance data for analysis
- Explore key trends using Exploratory Data Analysis (EDA)
- Summarize findings using Pivot Tables
- Visualize insights through an interactive Excel dashboard
- Identify factors influencing student success and failure

---

## 🛠️ Tools Used

| Tool | Purpose |
|---|---|
| Microsoft Excel | Data Cleaning, EDA, Pivot Tables, Dashboard |
| Excel Pivot Tables | Data Summarization & Aggregation |
| Excel Charts | Data Visualization |
| Excel Formulas | KPI Calculations & Metrics |

---

## 🔄 Project Workflow

### 1. 🧹 Data Cleaning
The raw dataset was cleaned to ensure accuracy and consistency before analysis:
- Removed duplicate records
- Standardized column headers and data formats
- Uppercased text
- Change type

---

### 2. 🔍 Exploratory Data Analysis (EDA)
After cleaning, the data was explored to understand distributions and relationships:
- Calculated descriptive statistics (mean, min, max) for Math, Science and English scores
- Analysed score distribution across final grades (A–F)
- Examined performance by gender and study method
- Investigated the relationship between study hours and overall score
- Identified top and bottom performing student segments

---

### 3. 📋 Pivot Table Summarization
Pivot Tables were used to aggregate and summarize the data across multiple dimensions:

| Pivot Table | Description |
|---|---|
| **Study Method Summary** | Count of students and average overall score by study method |
| **Top and Bottom Performers** | Count of students and sum of overall score by grade |
| **Gender Analysis** | Average overall score segmented by gender |
| **Performance Distribution** | Sum of scores across performance categories (Excellent, Good, Fail, Poor) |
| **Subject Performance by Grade** | Average Math, Science and English scores per final grade |

---

### 4. 📊 Data Visualization & Dashboard
An interactive Excel dashboard was built with the following charts:

| Chart | Type | Insight |
|---|---|---|
| **Subject Performance** | Clustered Bar Chart | Average scores per subject across grades A–F |
| **Gender Analysis** | 3D Pie Chart | Average overall score distribution by gender |
| **Study Hour vs Grade** | Scatter Chart | Relationship between study hours and total score |
| **Performance Distribution** | Bar Chart | Student count across performance categories |
| **Study Method Summary** | Pivot Table | Performance comparison by learning method |
| **Top and Bottom Performers** | Pivot Table | Grade-level performance breakdown |

**KPI Cards** were added at the top of the dashboard displaying:
- Total Students: **25K**
- Highest Score: **100**
- Average Score: **63.75**
- Pass Rate: **73%**
- Fail Rate: **27%**

---

## 💡 Key Insights

Out of 25,000 students, the average score is **63.75** with a pass rate of **74%**, meaning about **1 in 4 students is failing**. All genders perform almost equally at around **64 points**, showing no performance gap. Students who study more hours tend to score higher, confirming that **time investment is the biggest driver of success**. Among study methods, Online Videos produces slightly better results but the differences are minimal, suggesting **how students study matters less than how long they study**. The majority of students fall in the Good and Excellent performance categories, while higher grade students consistently outperform lower grade students across all three subjects.

> **Key Takeaway:** Performance is moderate overall, but the **27% fail rate** signals a need for early intervention and better support for struggling students.
